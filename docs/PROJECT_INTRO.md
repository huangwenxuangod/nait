# 指尖 SOP · Nail-It — 项目介绍

> 赛道：抖音精选-内容重构 | 方向：从"使用"出发  
> 作品形态：AI 美甲 DIY 助手 Web App  
> 体验地址：https://github.com/huangwenxuangod/nait

---

## 一、目标人群与痛点

**目标用户**：刷抖音/小红书种草美甲教程、想自己在家做但做不出来的个人玩家。

**核心痛点**：
- 教程看着简单，上手就废——步骤不清晰、细节没说透
- 每次做之前不知道要买什么材料，买回来发现缺东西
- 不确定这个款式适不适合自己的手型和肤色
- 左手给右手做完全不会，动作别扭效率低
- 做完没几天就掉了，不知道问题出在哪

---

## 二、解决方案

一条完整闭环：**看教程 → AI 拆解 → 虚拟试戴 → 物料准备 → 分步语音引导 → 拍照核验 → 生成社交视频**。

### 核心流程

```
用户刷到抖音美甲教程 → 复制链接
  → 粘贴到 App
  → AI（通义千问）自动拆解：
      · 这是什么款？什么颜色？
      · 需要哪些工具和色号？
      · 做这个要几步？每步怎么做？
  → 虚拟试戴：把款式"穿"到用户手上，对比不同甲型和长度
  → 物料清单：一键确认我有没有，缺的自动推荐替代品
  → 沉浸式制作：
      · 每步有视频示范 + AI 翻译后的子弹点指令
      · 倒计时 + 语音播报 + 拍照核验
      · 非惯用手模式（镜像提示）
  → 完成 → 自动生成抖音风格短视频 → 一键分享
```

---

## 三、核心创新点

### 1. 教程 → 可执行 SOP
不是简单总结视频，而是把模糊的原话（"薄薄涂一层就行"）翻译成精确指令（"刷头刮掉2/3，从甲根推到甲尖，两侧留0.5mm"），每步以子弹点呈现，让用户真的能跟着做。

### 2. 虚拟试戴 + 多甲型对比
解析出款式后，AI 把设计"穿"到用户手型轮廓上，支持 6 种甲型 + 3 档长度实时对比，用户确认后整个教程按选定甲型指导。

### 3. 个性化准备清单
不只是列工具，每项有两个按钮"我有/我没有"，点了"没有"自动弹出替代方案（"没有磁铁棒？可以用冰箱贴"），缺失项传入制作页全程提醒。

### 4. 分步语音引导 + 反馈闭环
每步有倒计时、语音播报、拍照核验（模拟 AI 检查是否做到位），最后一步完成后弹出护理提示（"做完2小时内别沾水"）和持久度提醒。

### 5. 从冷启动到社交分享
- 性别选择 → 男生/女生推荐不同风格
- 积分等级系统（完成越多等级越高）
- 完成自动生成抖音短视频 → 一键分享
